 # Clothes and Accessories: Sales and Product Details Dashboard

## Short Description
This project presents an interactive **Excell dashboard** that visualizes sales and product details for a retail dataset. It helps users analyze sales trends, product performance, and customer behavior across different regions and stores.

## Tech Stack
- **Excell Desktop** – Dashboard creation and visualization  
- **Excel / CSV** – Data source  

## Data Source
[Clothes and Accessories Sales Dataset](https://excelx.com/practice-data/sales-retail/)

## Features / Highlights
- **Problem / Goal:**  
  Retail managers often struggle to analyze multi-dimensional sales data efficiently. The goal of this dashboard is to **provide actionable insights** into product performance, sales by region, and customer behavior.
- Interactive **slicers** for filtering by `Region`, `Product`, and `Customer Type`.  
- **KPI Cards** for Total Sales, Total Orders, and Average Discount.  
- **Charts & Visuals:**  
  - Sales trend over time (Line Chart)  
  - Top-performing products (Bar Chart)  
  - Sales distribution by region (Column Chart)  
  - Payment method share (Pie Chart)  
- **Automation:** Calculated metrics updated dynamically using DAX.
